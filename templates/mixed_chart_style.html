# Mixed Chart Implementation Guide (Quality Control Dashboard) This guide provides the HTML and Tailwind CSS structure used to create the mixed chart (Columns for Defect Velocity and a Line for the Target) as seen in the "Digital Curator" design system. ## Visual Breakdown - **Columns (Bars):** Represent the actual "Defect Velocity" (e.g., daily defect rates). - **Target Line:** A horizontal reference line representing the 2.5% quality threshold. - **Legend:** Indicators for "RATE" and "TARGET (2.5%)". ## HTML & Tailwind CSS Snippet ```html

Defect Velocity & Target Chart

Defect resolution speed vs Last 30 Days

Rate
Target (2.5%)
D1 D5 D10 D15 D20 D25 D30
``` ## Implementation Notes 1. **Framework:** This uses **Tailwind CSS**. Ensure you have it configured in your project. 2. **Dynamic Data:** In a production environment, you would typically use a charting library like **Chart.js** or **Recharts**. To achieve this look in those libraries: - Set the dataset type for 'Rate' to `'bar'`. - Set the dataset type for 'Target' to `'line'`. - Use the hex color `#1e40af` (blue-800) for the bars and `#ef4444` (red-500) for the line. 3. **Typography:** The `font-manrope` class corresponds to the font defined in your design system.
